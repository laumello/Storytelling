# Visualización de Datos Demográficos con Principios Gestalt, Tufte y Holmes

## Descripción del Proyecto

Este proyecto tiene como objetivo aplicar principios de visualización de datos para comunicar información demográfica de manera clara y efectiva.

Utilizando datos de población por país, se desarrolla una historia visual centrada en la transformación demográfica mundial y el cambio de liderazgo poblacional entre India y China.

Las visualizaciones fueron diseñadas considerando recomendaciones de:

* Gestalt (percepción visual)
* Edward Tufte (claridad y eficiencia visual)
* Nigel Holmes (storytelling con datos)

El proyecto fue desarrollado en Python utilizando Plotly para crear gráficos interactivos y un dashboard ejecutivo.

---

## Objetivos

* Analizar la evolución de la población mundial.
* Identificar cambios en la distribución demográfica global.
* Aplicar principios de visualización para mejorar la comunicación de resultados.
* Construir una narrativa basada en datos.
* Diseñar un cuadro de control (dashboard) para facilitar la toma de decisiones.
